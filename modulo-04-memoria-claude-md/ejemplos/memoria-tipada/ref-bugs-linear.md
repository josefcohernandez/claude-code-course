type: reference
---

Los bugs del pipeline de ingestion se trackean en Linear, proyecto `INGEST`.

- URL: `https://linear.app/miempresa/team/INGEST`
- Etiqueta para bugs criticos: `P0`
- Responsable de triaje: equipo de data engineering
- SLA para incidencias `P0`: 4 horas de respuesta

Si aparece un bug del pipeline, derivar el seguimiento a Linear en lugar de abrir un issue
nuevo en GitHub.
