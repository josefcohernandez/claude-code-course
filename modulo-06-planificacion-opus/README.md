# Modulo 06: Plan Mode, Opus 4.6 y Workflows Eficientes

## Descripcion

Este modulo cubre las tecnicas avanzadas de planificacion con Claude Code:
Plan Mode para disenar antes de implementar, Opus 4.6 con razonamiento
adaptativo, y workflows probados para maximizar productividad.

## Objetivos

1. Dominar Plan Mode para disenar antes de ejecutar
2. Entender el razonamiento adaptativo de Opus 4.6
3. Saber cuando usar Opus vs Sonnet vs Haiku
4. Aplicar workflows probados al dia a dia
