# Módulo 06: Plan Mode, Opus 4.7 y Workflows

## Descripción general

Este módulo cubre las técnicas avanzadas de planificación con Claude Code: Plan Mode para diseñar antes de implementar, Opus 4.7 con razonamiento adaptativo y workflows probados para maximizar la productividad. También revisa el papel de Fast Mode y cuándo conviene combinarlo con Sonnet o con el backend acelerado de Opus 4.6.

## Objetivos

1. Dominar Plan Mode para diseñar antes de ejecutar.
2. Entender el razonamiento adaptativo de Opus 4.7 y el contexto de Fast Mode sobre Opus 4.6.
3. Saber cuándo usar Opus frente a Sonnet o Haiku.
4. Aplicar workflows probados al día a día.

## Duración

**2 h 15 min** (90 min de teoría + 45 min de ejercicios)

## Prerrequisitos

Módulos 01-05 completados.
